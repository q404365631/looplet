# Third-party users

Projects built on top of `openharness`. If you're shipping something
real with it, please send a PR or reply in the [Show and tell
discussion](https://github.com/hsaghir/openharness/discussions) — we'd
love to feature your work.

| Project | What it does | Scale / status |
| --- | --- | --- |
| [primal-security](https://github.com/hsaghir/security-primitives) | Autonomous security-operations agent with a six-primitive framework | Research / alpha |

## How to add your project

1. Open a PR that adds one row to the table above with:
   - A link to your project (OSS or public landing page).
   - A one-sentence description of what it does.
   - A rough maturity indicator (research / alpha / beta / production).
2. (Optional) Open a thread in the *Show and tell* discussion with a
   longer write-up — which backend you use, which hooks you wrote,
   anything you'd like feedback on.

We don't gate this on stars or popularity — if you're using it, you
belong on the list.
